# usecases/_shared

このディレクトリは、`usecases` ディレクトリ配下の複数のユースケース間で共有される基底の定義（DTO、インターフェース、共通ユーティリティなど）を配置する場所です。

## 依存ルール（依存方向の制限）

- **他のユースケース（`app/usecases/*`）は `_shared` に依存してよい。**
- **`_shared` は、他のどのユースケース（`app/usecases/*`）にも依存してはならない（水平・上方向への依存禁止）。**

このルールを守ることで、ユースケース間での不要な水平依存や循環参照を防ぎ、共通定義がクリーンな状態で保たれるように設計されています。
